Negrito vs Itálico.

Hoje vamos aprender um pouco mais sobre semântica!

Negrito

Podemos conseguir de duas formas:

tag+b (não semântico) = palavra.

STRONG+tag (semântico) = palavra

Itálico

Podemos conseguir de duas formas:

tag+i (não semântico) = palavra.

Em+tag = palavra.

Como marcar um texto?

Para marcar um texto usamos a tag Mark!

Letras grandes e pequenas.

Para usarmos letras grandes usamos o comando big (Não semântico)

Para usarmos letras minúsculas usamos o comando small.

Textos deletados e inseridos.

Para textos dele tados usamos o comando del.

Para textos inseridos usamos o comando ins.

Também temos a forma não semântica que não vamos utilizar que é o comando U.

Sobrescrito e subscrito

Para sobrescrever utilizamos o comando sup2.

Para subescrever utilizamos o comando sub2.
